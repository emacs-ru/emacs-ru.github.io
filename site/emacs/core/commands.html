Команды

В GNU Emacs большое количество команд, для запуска которых можно использовать привязанные к ним сочетания и последовательности клавиш. Ещё больше команд не имеют привязок к клавишам, однако, могут быть выполнены с помощью минибуфера.

Слова «выполните команду» следует понимать так:

  1. Нажмите M-x для перехода в минибуфер.

  2. Введите название команды.

  3. Нажмите RET.

  4. Если команда принимает аргументы, введите их, завершая ввод каждого аргумента нажатием RET.

Например, можно встретить выражение:

Включите режим whitespace-mode.

Это значит:

  1. Нажмите M-x для перехода в минибуфер.

  2. Введите команду whitespace-mode.

  3. Нажмите RET.
