Встроенные пакеты

В этом разделе представлены описания встроенных пакетов. Эти пакеты входят в состав GNU Emacs и доступны без установки.

Обновление

Возможно, вы захотите обновить встроенные пакеты. В этом случае делайте так:

  1. Выполните команду eval-expression, C-:.

  2. Введите выражение:

    (setopt package-install-upgrade-built-in t)

  3. Нажмите RET.

  4. Выполните команду list-packages.

    package.el покажет большое количество пакетов, которые можно обновить.

  5. Нажмите U, чтобы выделить все пакеты, но не нажимайте x.

  6. Отфильтруйте пакеты по статусу: / m.

  7. С помощью клавиши u снимите выделение со всех пакетов, кроме тех, которые нужно обновить.

    Как понять, что пакет нужно обновить? Нужно ответить "Да" на два вопроса:

    • Я знаю для чего нужен этот пакет?

    • Это обновление действительно необходимо?

    Без необходимости не обновляйте встроенные пакеты! Тем более не обновляйте пакеты, если не знаете, для чего они нужны!

  8. Нажмите x и подтвердите выполнение операций.
